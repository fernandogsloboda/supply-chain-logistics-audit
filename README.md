# 📊 Supply Chain Efficiency Audit & Inventory Optimization

## 📌 Executive Summary
This project performs a deep-dive diagnostic on a global supply chain dataset (+180k records) to identify logistics bottlenecks and financial risks. The analysis revealed that **over 54% of orders are delayed**, putting **$20.1 Million in revenue at risk**.

## 🚀 Key Business Insights
- **The "Premium" Failure:** The *First Class* shipping tier has a **95.3% late delivery rate**, scoring only **0.47/10** on the Efficiency Scorecard.
- **Revenue at Risk:** Identified $20.1M in sales currently affected by logistics inefficiencies.
- **ABC Inventory Strategy:** Classified products into A, B, and C categories, recommending a **2.2-day Safety Stock buffer** for high-priority items (A-Class) like Fishing and Cleats.
- **Seasonality:** Identified a massive demand spike in January, which currently overwhelms the existing carrier capacity.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** Pandas (Data Wrangling), Matplotlib/Seaborn (DataViz), NumPy (Analytics).
- **Dataset:** DataCo Smart Supply Chain (Big Data Analysis).

## 📈 Visualizations Included
1. **Carrier Performance Scorecard:** Ranking shipping modes from 0 to 10.
2. **Financial Loss by Category:** Quantifying canceled sales in dollars.
3. **Safety Stock Plan:** Tactical recommendations for inventory management.
4. **Logistics Risk Mapping:** Identifying critical failure points in the supply chain.

## 📂 Project Structure
- `supply_chain_analysis.ipynb`: Full Python code and documentation.
- `requirements.txt`: Necessary libraries to run the environment.
- `images/`: Exported charts for the final report.

## 🎯 Strategic Recommendation
Immediately renegotiate or terminate contracts with **First Class** carriers and shift volume to **Standard Class**, which proved to be 13x more reliable in this specific operation.
